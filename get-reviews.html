Get Reviews​

Use this API to retrieve all the reviews for a specific item.

Query Parameters
  • page number

    Default value: 1

    By default first page will be listed. For navigating through pages, use the page parameter.

  • limit number

    Default value: 20

    The number of results to return per request.

  • itemId number[] required

    The Unique identifier of the item.

  • createdAt number

    The date at which the review is created.

  • rating number

    The review score out of 5.

Responses
  • 200

Success response

  • application/json
